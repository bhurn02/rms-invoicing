QR Code Format

QR Code Contains:
Property Code

e.g., GCA

Unit Number

e.g., 101

Meter ID

e.g., M001

GCA|101|M001

Quick Troubleshooting

Camera Issues
  • Allow camera permissions
  • Check good lighting
  • Hold device steady
  • Clean QR code if dirty
Scanning Issues
  • Try different angles
  • Move closer/farther
  • Ensure QR code is flat
  • Check QR code quality

Offline Mode & Sync

Works Offline!

The system automatically saves readings offline when there's no internet connection.

  • Look for status dots on your avatar
  • Readings sync automatically when online
  • No data loss - everything is saved

Smart Sync Features
  • Connection stability check before sync
  • Manual sync button when needed
  • Progress indicators for sync status
  • Smart notifications for connection changes
Connection Notifications
  • Connection Lost: Red notification with two-line message
  • Connection Restored: Green notification confirming reconnection
  • Auto-dismiss: Notifications disappear automatically
  • Non-intrusive: Don't block your work flow

Pro Tips

Efficiency Tips

  • Add remarks for unusual readings
  • Keep device charged
Best Practices
  • Double-check meter readings
