Quick Start - 5 Steps

1. Start Scanner

Click "Start Scanner" button

2. Scan QR Code

Point camera at meter QR code

3. Enter Reading

Type current meter value

4. Submit

Click "Submit Reading"

Ready for Next Meter

QR Code Format

QR Code Contains:
Property Code

e.g., GCA

Unit Number

e.g., 101

Meter ID

e.g., M001

GCA|101|M001

Quick Troubleshooting

Camera Issues
  • Allow camera permissions
  • Check good lighting
  • Hold device steady
  • Clean QR code if dirty
Scanning Issues
  • Try different angles
  • Move closer/farther
  • Ensure QR code is flat
  • Check QR code quality
"Already Scanned"

This is normal! Duplicate prevention working correctly.

  • Check date shown in notification
  • Verify it's the same reading period
  • Scan next meter if correct
"Saved Offline" Badge

This is normal! Reading saved securely.

  • Reading is safe in local storage
  • Will sync when connection returns
  • Check orange dot on avatar

Data Accuracy Features

Duplicate Prevention

System automatically prevents duplicate readings for the same property/unit in the same period.

  • Instant Check: Validation happens immediately upon QR scan
  • "Already Scanned": Clear notification if duplicate detected
  • Form Hidden: Prevents accidental resubmission
  • Details Shown: See when and what value was previously recorded
Last Reading Validation

Last reading displayed prominently above current reading input for easy validation.

  • Large Display: Last reading shown in large, bold text
  • Date Shown: See when the last reading was taken
  • Usage Calc: Previous reading and usage displayed
  • No Scrolling: Everything visible without scrolling

Offline Mode & Sync

Works Offline!

The system automatically saves readings offline when there's no internet connection.

  • Look for status dots on your avatar
  • Readings sync automatically when online
  • No data loss - everything is saved
  • NEW: Offline readings visible immediately in Recent Readings table
Green Dot

Online & Synced

Red Dot

Offline

Orange Dot

Pending Sync

Smart Sync Features
  • Connection stability check before sync
  • Manual sync button when needed
  • Progress indicators for sync status
  • Smart notifications for connection changes
Connection Notifications
  • Connection Lost: Red notification with two-line message
  • Connection Restored: Green notification confirming reconnection
  • Auto-dismiss: Notifications disappear automatically
  • Non-intrusive: Don't block your work flow
Recent Readings Table

NEW: See all readings (online and offline) in one table.

  • "Saved Offline": Orange badge for offline readings
  • "Synced": Green badge after successful sync
  • Complete Info: Tenant name, property, reading value, timestamp
  • Auto-Update: Table refreshes after sync completion

Pro Tips

Efficiency Tips
  • Scan QR codes in sequence
  • Use "Recent Readings" to verify
  • Add remarks for unusual readings
  • Keep device charged
Best Practices
  • Double-check meter readings
  • Report damaged QR codes
  • Use consistent reading format
  • Take photos if needed
